サポーター向けマニュアル-引用ポスト

いつもPlayers1st&nbsp;で選手のポストを楽しみにしている、スポーツファンの皆さんへ。 スポーツ選手が投稿したポストやイベント、また試合結果などにコメントが投稿できる“引用ポスト”機能をリリースしました！ 選手の投稿に感想

Feb 01, 2017

いつもPlayers1st で選手のポストを楽しみにしている、スポーツファンの皆さんへ。

スポーツ選手が投稿したポストやイベント、また試合結果などにコメントが投稿できる“引用ポスト”機能をリリースしました！

選手の投稿に感想や応援を書くことができます🙆 

ぜひ、皆さんの熱いエールをお書きください💪

では、その方法を一つ一つご紹介します。

引用ポスト💬の作り方

まず、例として選手のポストを一つ開きます。

（下の図は架空の選手のポストです）


一番下にある「この内容を引用してポストする」のリンクを押すと、ポスト作成の画面に移ります。


選手のポストと同様に、ハッシュタグをつけて、同じタグをつけたポストとつながることもできます。

画像📷か動画📹（どちらか１つ）を添付することもできます。


FacebookやTwitterに連携すると、引用ポストと同時に連携したSNSにも投稿されます。

最後に「登録する」ボタンを押すことをお忘れなく!!

引用ポストはこのように表示されます。

この例では、スポーツ選手のポストへの引用ですが、イベントや試合（現在はFリーグのみ提供中）に対しての引用ポストもできます🙆

誰に見られるの？

この引用ポストは、Players1st上で自分のことをフォローしている人のタイムラインに流れます。

また、サポーターの詳細ページにも引用ポストの一覧が掲載されます。

ハッシュタグをつけた場合は、そのタグをたどった人にも見られます。
例えば、同じスポーツ名・チーム名のタグを経由してサポーターの同士を発見することもできます。

もちろん、SNSとの連携・シェアをした場合は、そのSNS上のフォロワー・友達にも流れます。

Twitterに連携・シェアした場合は、自動的にPlayers1st公式アカウントがリツイート（RT）もします。

どんなことを書けばいい？

基本的に引用ポストに制限はありません。

選手・チームへの応援、観戦記、紹介など、その引用元に関わるポストを歓迎します。

お互いにスポーツ選手、スポーツファンがチームやスポーツのジャンルを超えて行き来ができることがPlayers1stの特長です。

対戦相手がいる競技の場合、そのチーム、選手に対してのネガティブなポストをされると、応援しているチーム・競技に悪いイメージを持たれてしまうことにつながってしまいます。
