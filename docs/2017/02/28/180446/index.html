Players1stはより良く使っていただくために、便利な機能をリリースし続けています。

今回は、シンプルだけど嬉しいあの機能。

スポンサー募集の支払い方法に銀行振込が増えました！！🏧

今まで選手の作るスポンサー募集への支払い方法は、PayPalによるクレジット決済のみでした。

  • PayPalアカウント持っていないし
  • クレジットカードは使いたくなんだけど
  • 年一括分でもいいから銀行振込がいいんだけど

のような声🗣をいただいていました。そのような要望にしっかり答えるために開発しました💻

さっそく、紹介していきましょう。

支払方法が毎月ならクレジット、年一括なら銀行振込

今までのクレジット決済に加えて、銀行振込が選択できるようになっています。

ただし、支払える期間は年一括のみとなっています。

ですので、

  • 毎月支払いたい場合はクレジット決済💳
  • 年一括なら銀行振込🏧

と支払いたい期間によって選択することになります。

年一括はシーズン更新月まで分を支払い

年一括は選手のシーズンが終わるまでの残りの月数分までとなります。 スポンサー募集の画面では更新月という表示で確認できます。

シーズン初めの月にスポンサー募集を応募した場合は12ヶ月分となりますが、 シーズン途中、例えば、毎年3月が更新確認のスポンサー募集に対して12月に応募したら、12月,1月,2月,3月の4ヶ月分の金額が年一括の銀行振込で支払う金額となります。

ちなみに、選手のスポンサー応募にはユーザー登録(無料)が必要です。まだアカウント登録したことがない方は こちら からお願いします。

では、年一括の支払い方法をご紹介しましょう。

年一括の支払い方

まず、スポンサー募集一覧から支払いたいスポンサー募集を選び、「詳細を見る」リンクをタップします。

個別スポンサー募集のページが開きますので、下の方の「応募する」リンクをタップします。

応募内容にて、毎月か年一括かを選択肢できます。銀行振込の場合は「年一括」を選択します。

口数に数字を入力し、スポンサー紹介に表示しない場合はチェックを付けて、「応募する」ボタンをタップします。

そうすると、受け付けられた旨のメッセージが表示され、登録メールアドレス宛にメールが送られます📩

あとは受け取ったメールにしたがって、振込金額や振込先の情報を見ながら振込処理をしていただくことになります👌

また、振込期限は応募日から1週間以内、振込手数料は支払い者の負担にてお願いします。

やはり定番の銀行振込😊

いかがだったでしょう。操作は非常にシンプル。「年一括」を選択するのみです。

クレジット決済でのインターネットでの支払いは増えましたが、まだまだ銀行振込は根強い支払い方法ですよね。

これを機に、クレジットカードをお持ちでなかった方は応援したい選手のスポンサー応募を検討してみてはいかがでしょう。お待ちしています！
