新機能

スポンサー募集のステータスができました

選手から要望いただいていたスポンサー募集のステータス機能をリリースしました🎉 これによって、スポンサー募集の状態を公開・非公開・応募終了の3つに設定することができ、より便利な使い方ができます👍 さっそく見ていきましょう。 ステータスとは、スポンサー募集を以下の3つの状態に設定します。 公開 非公開 応募終了 公開 今までのスポンサー募集の状態と同じで、誰からも見れて応募を受け付けている状態です。 今までのスポンサー募集はこの公開の状態1つしか選べませんでした。
