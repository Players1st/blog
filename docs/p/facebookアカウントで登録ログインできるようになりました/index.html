Jul 19, 2017

アカウント作成でメールアドレス、パスワード、プロフィール文などを入力するのは手間ですよね。

Players1stでは、プロフィールに様々な入力項目がありますが、なかなか全てを埋めていただけるのは難しいようです😅

選手にとってはサポーターから見つけやすくしてくれるための大事な項目でもあるので、是非とも入力していただきたいところ。

そこで、Facebookアカウントから簡単にアカウント登録できるようにしました‼️ 自動的にFacebookに登録されているプロフィール情報からPlayers1stで利用する情報を取得し保存します。

数クリックで終わるので本当に簡単です。さっそくFacebookアカウントでの登録方法を見てみましょう。

Facebookアカウントで選手登録する方法

Players1st代表の深海のFacebookアカウントを使って行ってみます。

選手が登録する方法を書いていますが、選手を支えるサポーターの立場の方もサポーター登録画面から同じようにFacebookアカウントで登録が行えます。

登録を考えてる方も一緒に手順通り進めば必ずできますので、これを機に是非登録してみましょう。

Players1stトップページからスタート

まず、トップページから選手登録ボタンを押します。

すると、「Facebookアカウントで登録する」というボタンがあります。これをクリックします。



Facebookに移動

FacebookのWebページが表示され、Players1stが各データを受け取っていいかどうか同意を求めれますので、ログインボタンを押します。

次に、Players1stがFacebookのタイムラインに書き込む際の公開範囲を設定します。

デフォルトでは「友達」が選択されている状態かと思います。 実際にPlayers1stが勝手にFacebookのタイムラインに書き込むことはありませんのでご安心ください。

不安な方は「自分のみ」を選択されることをおすすめします。

選択したらOKを押します。

Players1stで追加情報を入力して完了

Players1stの画面に戻り情報変更画面が表示されます。そこにはすでに、プロフィール画像、プロフィール、活動拠点、出身地、出身校などの情報が入力されています。

選手のみなさんは、「競技カテゴリー」「所属チーム」など大事な情報を入力しましょう。 「身長」「体重」「血液型」も入力できるといいですね👍

最後に一番下の「Update」ボタンを押せば登録完了です🎊

いかがでしょう？

入力の手間がなくせてとても簡単でしたね😁

Facebookアカウントでログインする方法

Facebookアカウントで作成した場合は、ログインもFacebookアカウントで行います。

まず、ログイン画面にアクセスすると、「Facebookアカウントでログイン」というボタンがあります。これをクリックします。

すると、「Facebookアカウントによる認証に成功しました。」というメッセージが表示されて、ログインできます。

ワンクリックでログイン。本当に簡単です😆

今後も便利な機能を開発中

Players1stでは、このように選手やサポーターにとって使いやすい機能を日々開発しています。 もし、わからないことがあったり、もっとこうしてほしいなどの要望があれば是非教えてください。

Players1stのLINEアカウントもしくは問い合わせフォームどちらからでも受付けています。

Players1st は、「スポーツの力で世の中をより良くする」をミッションにアスリートのみなさんの経済的課題を解決するため、選手とサポーターを継続的につなげる課金PRサイトを運営しています。

「スポーツを支える」力になりたい。そんな思いでチームは日々開発しています。
