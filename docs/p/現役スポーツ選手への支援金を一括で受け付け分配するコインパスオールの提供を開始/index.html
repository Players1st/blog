現役スポーツ選手への支援金を一括で受け付け分配する「コインパスオール」の提供を開始

「 現役スポーツ選手を見つける、つながる、支える。 」をミッションに掲げる株式会社プレイヤーズファースト (本社: 東京都世田谷区、代表取締役社長 : 深海 寛信、以下プレイヤーズファースト) は、「Players1st (プレイヤーズファースト) (URL : https://players1.st)」において、支援金を一括で受け付け、現役スポーツ選手に分配する「コインパスオール」を2017年2月14日に開始します。

現役トップのスポーツ選手に広く届ける

プレイヤーズファーストでは、現役トップレベルのスポーツ選手の支援サービスとして、2016年8月にオープン、同年9月にはインターネット上から特定の選手へ支援金を届ける「コインパス」の提供を開始し、選手へ支援金を届けてきました。

「コインパスオール」は、Players1stに登録する現役トップレベルのスポーツ選手全員に支援金を届けることのできる、支援金一括受付サービスです。支援金を渡す選手の範囲が広がり、広く支援することができます。

「コインパスオール」は企業の社会貢献活動や企業のCSR活動に利用いただけることを想定しています。選手・競技種目に偏らないため、全ての競技種目のスポーツ選手を支援できることになります。Players1stをかえすことで支援範囲の拡大、現役トップレベル選手の支援を実現できます。

支払い方法として、従来のクレジット決済に加え、請求書による銀行振込にも対応、企業からの支援を受け付けやすくしました。受付金額は20万円からを想定しています。

Players1stにて支援金を受付、手数料を差し引いた金額をPlayers1st全登録選手へ分配します。手数料は現在15%、残りの85%を全選手に分配します。手数料にはシステム運用費、選手の口座への振込手数料などを含みます。

お問い合せ・支援の受付は、Players1stお問い合わせフォーム(URL : https://players1.st/feedbacks/new)またはメール contact@players1.st にて受け付けています。

背景

近年の企業の社会貢献活動やCSR活動はは活発となっており、企業の社会的活動への参加が求められています。しかしながら、その活動がスポーツの分野の支援が行き渡らない、とりわけ選手自身への支援として届かないなどの課題がありました。また、選手自身も支援金を受け付ける課金窓口を設け、自ら運営することが難しいという課題もありました。

登録選手について

現在、国内フットサル最高峰リーグ Fリーグのトップチーム選手や国内最多優勝回数のビーチサッカークラブの選手、国内海外の試合に参戦している選手、2020年東京オリンピック日本代表を目指す選手など国内トップレベルのスポーツ選手が利用しています。 なりすまし防止のための本人確認、活動実績、参戦している試合・リーグのレベルなどをチェックした後、選手として登録しています。
